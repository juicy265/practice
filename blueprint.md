# 오늘 뭐 먹지? - 점심 메뉴 추천 사이트

## 개요

직장인들의 영원한 숙제, "오늘 점심 뭐 먹지?"에 대한 고민을 해결해주는 간단한 웹 애플리케이션입니다. 버튼을 누르면 랜덤으로 점심 메뉴를 추천해주고 해당 음식의 사진을 보여줍니다.

## 디자인 및 기능

*   **메인 페이지:**
    *   중앙에 "오늘 뭐 먹지?"라는 제목과 세련된 카드 레이아웃이 표시됩니다.
    *   "메뉴 추천받기" 버튼(둥근 모서리와 그림자 효과)이 있습니다.
    *   결과 영역: 추천 메뉴 이름과 함께 해당 음식의 이미지가 부드럽게 나타납니다.
*   **스타일:**
    *   모던한 그라데이션 배경과 화이트 카드 디자인을 적용했습니다.
    *   반응형 레이아웃으로 모바일에서도 잘 보입니다.
    *   `transition`과 `opacity` 효과를 사용하여 이미지가 자연스럽게 로드되도록 구현했습니다.
*   **기능:**
    *   랜덤 메뉴 선택: 9가지 인기 점심 메뉴 중 하나를 무작위로 선택합니다.
    *   이미지 표시: 각 메뉴에 매칭되는 고화질 이미지를 Unsplash API를 통해 가져와 표시합니다.
    *   로딩 상태: 추천 버튼 클릭 시 짧은 지연 시간과 함께 "추천 중..." 상태를 표시하여 사용자 경험을 향상시켰습니다.

## 구현 상세

*   `index.html`: 결과 레이아웃 구조 업데이트 (이미지 컨테이너 추가).
*   `style.css`: 카드 디자인, 버튼 스타일, 이미지 애니메이션 추가.
*   `main.js`: 메뉴 데이터 구조화 (객체 배열), 이미지 로딩 로직 및 UI 인터랙션 구현.
