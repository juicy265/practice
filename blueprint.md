# 오늘 뭐 먹지? - 점심 메뉴 추천 사이트

## 개요

직장인들의 영원한 숙제, "오늘 점심 뭐 먹지?"에 대한 고민을 해결해주는 간단한 웹 애플리케이션입니다. 버튼을 누르면 랜덤으로 점심 메뉴를 추천해줍니다.

## 디자인 및 기능

*   **메인 페이지:**
    *   중앙에 "오늘 뭐 먹지?"라는 큰 제목이 표시됩니다.
    *   "메뉴 추천받기" 버튼이 있습니다.
    *   버튼을 누르면 추천 메뉴가 표시될 영역이 있습니다.
*   **스타일:**
    *   모던하고 깔끔한 디자인을 적용합니다.
    *   가독성 좋은 폰트를 사용합니다.
    *   부드러운 애니메이션 효과를 추가하여 사용자 경험을 향상시킵니다.
*   **기능:**
    *   "메뉴 추천받기" 버튼을 클릭하면 미리 정의된 메뉴 리스트에서 랜덤으로 하나를 선택하여 보여줍니다.

## 현재 요청된 변경 사항

*   `index.html` 파일을 직장인 점심 메뉴 추천 사이트로 변경합니다.
*   `style.css`와 `main.js` 파일도 함께 업데이트하여 기능과 디자인을 구현합니다.
